# Find SealMetrics' Account ID

Your Account ID appears in several sections under:

- **Common tracker**  
- **1st party tracker**  
- **E-commerce plugins**

<img src="/img/account-setup/find-account-id.png" alt="Find account ID screenshot"/>

Open **Setup → Tags & API Info**, and you’ll see the Account ID displayed inside each tracker section.
